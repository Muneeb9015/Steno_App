Operational Manual

PROTOCOL V1.0 // USER GUIDE

PHASE 1: ENCRYPTION

01 Upload Cover Image: Select a high-quality PNG or JPG image. This will serve as the "container" for your secret.

02 Input Data: Type your secret message into the payload field.

03 Set Key: Enter a strong password. You MUST remember this password to decrypt later.

04 Execute: Click "Encrypt" and download the generated file.

PHASE 2: TRANSMISSION (CRITICAL)
DATA CORRUPTION WARNING

Standard messaging apps (WhatsApp, Messenger, Discord) compress images to save space. Compression destroys the hidden data.

CORRECT WAY TO SEND:

  • WhatsApp: Send as "Document" (Paperclip > Document)
  • Email: Send as Attachment
  • Discord: Send as File (or Zip it first)
PHASE 3: DECRYPTION

01 Upload Asset: Select the secret image you received.

02 Authenticate: Enter the exact password used during encryption.

03 Extract: Click "Decrypt" to reveal the hidden text.
